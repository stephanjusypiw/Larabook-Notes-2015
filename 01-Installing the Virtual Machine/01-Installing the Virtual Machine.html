2014-Installaing-Xampp-1.8.3



This tutorial will demonstrate how you can install Xampp 1.8.3 (at the time or writing).


Navigate to the URL shown below.  I if it best to download the ZIP file of Xampp so I can run it from my external hard drive.




Save the ZIP file on the Desktop.





I will unzip Xampp on to the root folder of my external hard drive.  YOU MUST UNZIP XAMPP ONTO A ROOT FOLDER.  Either
C:\ or D:\  etc.




The unzipped Xampp. 

First, click the setup_xampp file and follow the instructions.  This will confgure Xampp for your
hard drive


Let's now open the Xampp control, as shown below.



Select USA





Let's start MySQL.

NOTE:  You may get a Windows Firewall message, in which case you must accept the service.




The MySQL database server




Now start Apache.

NOTE:  You may get a Windows Firewall message, in which case you must accept the service.




Open a browser and poin to the URL below.  Select English.




The Xampp home page.




If you click phpinfo(), you can see the current version of PHP on your system, in this case 5.5.3.




Let's now administer MySQL by using phpMyAdmin.




This is where you can administer the MySQL database server.  The first thing we need to do is set the root assword for the MySQL server.



Back at the home page select Security.




Now select the URL link below.



Type a password for the MySQL server.  DO NOT FORGET IT!




The MySQL root password has been changed.




Stop the MySQL server.




Start the MySQL server.



Enter phpMyAdmin once again after you have refreshed the web page.




You are now password protected, so type the MySQL password



You are in.
